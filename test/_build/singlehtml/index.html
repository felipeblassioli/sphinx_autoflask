GET /product/best/¶

Returns a set of recommended products based on the parameter.

Request JSON Object:
 
  • product_id (int) – None.
  • limit (int) – (optional) max products count.. *Default*=5
POST /product/map/¶

Returns a set of nearest stores’s geocod_data with product quantity in each stores for each product in array ‘ids’.

Request JSON Object:
 
  • ids (unicode) – None.
  • location (object) – (optional) {“lat”: double, “lng”: double} . Position for which nearest stores are calculated. If none is given, returns all stores.. *Default*={‘lat’: None, ‘lng’: None}
  • radius (float) – (optional) Returns only stores within given radius. Location must be specified.. *Default*=10.0
  • limit (int) – (optional) Returns an array with maximum row count given by limit. In this case, location must be specified.. *Default*=None

POST /login/¶

Logs In the user and returns a session token which is used on other services. See Authentication.

Request JSON Object:
 
  • uid (unicode) – unique identification of user mobile device (Android Serial Number / MAC address).
  • cpf (unicode) – (optional) None. *Default*=None
  • facebook_data (object) – (optional) jsonObject: {access_token}. *Default*=None
